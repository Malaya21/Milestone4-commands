# UERANSIM Commands

## Network Function Simulator Commands

### Start UE (User Equipment)
```bash
cd /UERANSIM/build
sudo ./nr-ue -c ../config/open5gs-ue.yaml
```

### Start gNB (Base Station)
```bash
cd /UERANSIM/build
./nr-gnb -c ../config/open5gs-gnb.yaml &
```

---

## Combined Setup

Run the gNB in the background first, then start the UE:

```bash
# Terminal 1: Start gNB
cd /UERANSIM/build
./nr-gnb -c ../config/open5gs-gnb.yaml &

# Terminal 2: Start UE
cd /UERANSIM/build
sudo ./nr-ue -c ../config/open5gs-ue.yaml
```

---

## Notes

- **gNB** (5G Base Station) should be started before UE
- The `&` at the end of gNB command runs it in background
- **UE** (User Equipment) requires `sudo` for network privileges
- Configuration files located in: `../config/`
  - gNB config: `open5gs-gnb.yaml`
  - UE config: `open5gs-ue.yaml`
